RSV Gmaps is plugin for google maps you can add Multiple Pointers in one Maps.

== Description ==

RSV GMaps this Plugin for displaying Multiple Pointers in Google Maps. It will produce clean HTML code without any iframe.

== Installation ==

1. Upload the <strong>rs-gmaps</strong> folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the '<strong>Plugins</strong>' menu in WordPress
3. Use This Short Code to Display All pointers: <strong>[RSV_GMaps]</strong>
4. Use This Short Code to Display Single Pointer: <strong>[RSV_GMaps id=16]</strong>
5. Use This Short Code to Display Multiple Pointers: <strong>[RSV_GMaps id=16,20,36]</strong>.
4. You can change <strong>id</strong> for every shortcode used. 
5. Go to Rsv Gmaps in your WordPress admin, and click the location you need the ID of. Then, if you look in the address bar of your browser, you'll see something like this: 
<strong>http://localhost/wordpress/wp-admin/post.php?post=31&action=edit</strong>
in this case 31, is the post ID.


== Frequently Asked Questions ==

= How do I insert the RSV GMaps? =
You can insert the RSV GMaps by pasting the shortcode `[RSV_GMaps]` into the content area of a page or post.  **Be sure to use the HTML editor when inserting shortcodes!** To customize your MAP, there are optional attributes that can be used with the shortcode.

= where can find the 'post ID' ? =
Go to Rsv Gmaps in your WordPress admin, and click the location you need the ID of. Then, if you look in the address bar of your browser, you'll see something like this: 
<strong>http://localhost/wordpress/wp-admin/post.php?post=31&action=edit</strong>
in this case 31, is the post ID.
